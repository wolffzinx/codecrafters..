Estruturas de Controle em C

Em C, usamos estruturas de controle para tomar decisões:

if (condição) {
    // código se verdadeiro
} else {
    // código se falso
}

// Exemplo:
if (idade >= 18) {
    printf("Maior de idade");
} else {
    printf("Menor de idade");
}
