Funções em C

O que são Funções?

Funções são blocos de código que realizam tarefas específicas. Elas ajudam a organizar e reutilizar código.

Estrutura de uma Função:

Função sem retorno

Usa void quando não retorna valor

void saudacao() { printf("Olá!"); }

Função com retorno

Especifica o tipo do retorno

int soma(int a, int b) { return a + b; }

Função com parâmetros

Recebe dados para processar

float media(float n1, float n2) { return (n1 + n2) / 2; }

Chamada de função

Como usar a função

int resultado = soma(5, 3); printf("%d", resultado);

Regras Importantes:

  • Declare a função antes de usá-la
  • O tipo de retorno deve corresponder ao declarado
  • Parâmetros são opcionais
  • Use return para retornar valores
  • Funções void não precisam de return

Exemplos Práticos:

// Função que calcula área do retângulo
float calcularArea(float base, float altura) {
    return base * altura;
}

// Função que imprime mensagem
void mostrarMenu() {
    printf("1. Iniciar\n");
    printf("2. Configurar\n");
    printf("3. Sair\n");
}

// Função que verifica número par
int ehPar(int numero) {
    return numero % 2 == 0;
}

// Como usar:
float area = calcularArea(10.0, 5.0);
mostrarMenu();
if (ehPar(4)) {
    printf("É par!");
}

Dica: Funções tornam seu código mais organizado e reutilizável. Use nomes descritivos que indiquem claramente o que a função faz!
