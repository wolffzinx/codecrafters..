Variáveis e Tipos em C

O que são Variáveis?

Variáveis são espaços na memória onde guardamos informações. Pense nelas como caixas etiquetadas onde podemos guardar diferentes tipos de dados.

Principais Tipos em C:

int

Números inteiros

int idade = 25;

float

Números decimais

float peso = 70.5;

char

Caracteres únicos

char letra = 'A';

double

Decimais precisos

double pi = 3.14159;

Regras Importantes:

  • Toda variável precisa ter um tipo definido
  • O nome não pode começar com número
  • Não use espaços no nome da variável
  • Declare a variável antes de usá-la
  • Use nomes significativos para suas variáveis

Exemplos Práticos:

// Declaração e inicialização de variáveis
int numero = 10;          // Número inteiro
float altura = 1.75;      // Número decimal
char inicial = 'J';       // Caractere único
double preco = 99.99;     // Decimal preciso

// Exemplos de nomes válidos
int idade;
float mediaFinal;
char primeiraLetra;
double salarioBase;

// Exemplos de nomes inválidos
int 1numero;      // Não pode começar com número
float media final; // Não pode ter espaço
char @letra;      // Não pode usar caracteres especiais

Dica: O tipo da variável determina que tipo de dado ela pode guardar e quanto espaço ocupa na memória. Escolha sempre o tipo mais adequado para sua necessidade!
