Listas e Dicionários em Python

Listas:

Listas são coleções ordenadas de itens que podem ser de diferentes tipos.

# Criando listas
frutas = ['maçã', 'banana', 'laranja']
numeros = [1, 2, 3, 4, 5]

# Acessando elementos
primeira_fruta = frutas[0]      # maçã
ultima_fruta = frutas[-1]       # laranja

# Modificando elementos
frutas[1] = 'pera'              # ['maçã', 'pera', 'laranja']

# Métodos comuns
frutas.append('uva')            # Adiciona ao final
frutas.remove('maçã')           # Remove elemento
len(frutas)                     # Tamanho da lista

Dicionários:

Dicionários são coleções de pares chave-valor.

# Criando dicionários
pessoa = {
    'nome': 'Ana',
    'idade': 25,
    'cidade': 'São Paulo'
}

# Acessando valores
nome = pessoa['nome']           # Ana
idade = pessoa.get('idade')     # 25

# Modificando valores
pessoa['idade'] = 26

# Métodos comuns
pessoa.keys()                   # Lista de chaves
pessoa.values()                 # Lista de valores
pessoa.items()                  # Lista de pares chave-valor
