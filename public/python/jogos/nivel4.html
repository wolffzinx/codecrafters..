Funções e Módulos em Python

Funções são blocos de código reutilizáveis que executam uma tarefa específica. Módulos são arquivos Python que contêm funções, variáveis e classes que podemos usar em outros programas.

Definindo Funções:

def saudacao(nome):
    return f"Olá, {nome}!"

# Chamando a função
mensagem = saudacao("Ana")
print(mensagem)  # Saída: Olá, Ana!

Módulos Comuns:

  • math: Funções matemáticas (sqrt, cos, sin)
  • random: Geração de números aleatórios
  • datetime: Manipulação de datas e horas

Importando Módulos:

import math
raiz = math.sqrt(16)  # 4.0

from random import randint
numero = randint(1, 10)  # número aleatório entre 1 e 10
