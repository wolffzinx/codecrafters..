Variáveis e Tipos em Python

Em Python, variáveis são como caixas onde guardamos diferentes tipos de informação. Não precisamos declarar o tipo da variável - Python faz isso automaticamente!

Tipos Principais:

  • str (string): Texto entre aspas - "Olá" ou 'Python'
  • int (inteiro): Números inteiros - 42, -10, 0
  • float (decimal): Números com decimal - 3.14, -0.001
  • bool (booleano): True ou False

Exemplos:

nome = "Ana"          # string
idade = 25           # inteiro
altura = 1.65        # float
python_legal = True  # booleano

Para verificar o tipo de uma variável, use: type(variavel)
