Collections e Exceptions em Java

Collections

Collections são estruturas que armazenam e organizam dados. As principais são:

  • ArrayList: Lista dinâmica que pode crescer/diminuir 
    ArrayList lista = new ArrayList<>();
lista.add("item");
  • HashSet: Conjunto que não permite duplicatas 
    HashSet conjunto = new HashSet<>();
conjunto.add(1);
  • HashMap: Armazena pares chave-valor 
    HashMap mapa = new HashMap<>();
mapa.put("chave", 1);

Exceptions

Exceptions são erros que podem ocorrer durante a execução. Tratamos com try-catch:

try {
    // código que pode gerar erro
} catch (Exception e) {
    // tratamento do erro
}

Principais exceptions:

  • NullPointerException: Ao tentar acessar objeto null
  • ArrayIndexOutOfBoundsException: Índice inválido em array
  • NumberFormatException: Erro ao converter String para número
