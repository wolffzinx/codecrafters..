Variáveis e Tipos em Java

Em Java, toda variável tem um tipo específico que determina que tipo de dado ela pode armazenar. Os tipos mais comuns são:

  • int: números inteiros (ex: 1, -5, 1000)
  • double: números decimais (ex: 3.14, -2.5)
  • String: texto (ex: "Olá mundo")
  • boolean: verdadeiro ou falso (true/false)
  • char: caractere único (ex: 'a', '1', '$')

Para declarar uma variável, use a sintaxe: tipo nomeVariavel = valor;

Exemplos:

int idade = 25;
double altura = 1.75;
String nome = "João";
boolean estudante = true;
char letra = 'A';

Lembre-se:

  • Strings usam aspas duplas (" ")
  • char usa aspas simples (' ')
  • Números decimais usam ponto, não vírgula
  • Nomes de variáveis não podem começar com números
